# Alberta Anglers Guide — Knowledge Base

Add `.txt` files here. They will be chunked and embedded when you run `load_kb.py`.

## Suggested files

- `regulations_overview.txt` — General Alberta sportfishing rules
- `zones_and_waters.txt` — Zone descriptions (ES1, ES2, PP1, etc.)
- `species_guide.txt` — Per-species seasons, limits, methods
- `stocking_schedule.txt` — ACA stocking data (update annually / seasonally)
- `seasonal_calendar.txt` — Month-by-month what's biting
- `geolocation_zones.txt` — Water body → zone mapping

## Format

Plain text, clear sections:

```
RAINBOW TROUT — EDMONTON SOUTH ZONE (ES2)
==========================================

Season: Year-round on most stocked ponds.
Limit: 5 per day.
Minimum size: None on stocked ponds.
...
```
